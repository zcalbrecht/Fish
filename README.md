# Interactive Fish Pond Simulation

A beautiful, interactive web-based pond simulation featuring fish, lily pads, duckweed, flowers, and dragonflies. Click or tap to interact with the ecosystem!

## Features

- **Interactive Fish**: Colorful fish that swim autonomously and respond to clicks/taps
- **Draggable Elements**: Move lily pads, flowers, duckweed clusters, and a raft around the pond
- **Dynamic Ecosystem**: 
  - Lily pads with stems that sway in the water
  - Duckweed clusters that drift and wobble
  - Flowers that bloom near lily pads
  - Dragonflies that fly across the screen
  - Ripples created on interaction
- **Responsive Design**: Scales beautifully across different screen sizes
- **Smooth Animations**: Spring physics and fluid motion throughout

## How to Use

1. Open `index.html` in a modern web browser
2. Click or tap anywhere to create ripples and attract fish
3. Drag lily pads, flowers, duckweed, or the raft to move them around
4. Watch the ecosystem come to life with autonomous behaviors

## Technical Details

- Pure JavaScript (no dependencies)
- HTML5 Canvas for rendering
- Responsive scaling system for different screen sizes
- Physics-based animations with spring systems
- Collision detection for surface items

## Browser Support

Works best in modern browsers that support:
- HTML5 Canvas
- ES6+ JavaScript features
- CSS3
